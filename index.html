About

I started out in computational biology at TGen, building the sequencing pipelines and mutation-detection tools used in cancer research. That turned into several years at Xamarin and then Microsoft working on the Mono runtime — the open-source .NET implementation that Xamarin was built on. More recently, I led the OS team at Wilderness Labs, where I built an embedded operating system from scratch that runs .NET on microcontrollers.

I have a B.S. in computer science from Arizona State University and have co-authored 41+ peer-reviewed papers in genomics, cited by over 11,000 researchers. My GitHub is @alexischr.
